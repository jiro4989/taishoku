== development environment

 go version go1.11 linux/amd64

== usacase

.共同作業者に退職の意図を伝える
[source,bash]
taishoku | wall

.唐突に職場を離れることになったときに、次回作業者に退職の意図を伝える
[source,bash]
echo taishoku >> ~/.bashrc

.そのまま使う
[source,bash]
taishoku --your-name 退職太郎 -y 2019 -m 1 -d 1
